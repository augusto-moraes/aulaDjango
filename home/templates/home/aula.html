Coisas Pra Lembrar

Como eu insalo o Django na minha máquina?

Considerando que você ja tem python na sua máquina, apenas digite o seguinte comando no terminal:

$: sudo pip install django

Como criar um projeto?

Primeiro crie um diretório para o Django e entre nesse diretório:

$: mkdir Django

$: cd Django

Depois inicie seu projeto:

$: django-admin startproject "nome do projeto"

Entre no diretório que será criado e abra o projeto no seu editor de texto:

$: cd "nome do projeto"

$: atom .

Para rodar o servidor, execute o arquivo manage.py com o argumento runserver:

$: python manage.py runserver

O que é Bootstrap?

Bootstrap é um framework util e gratuito de CSS e JS, que foi utilizado, por exemplo, nessa aplicação.

O que é Jinja?

Jinja é um framework que consegue relacionar seus códigos. Essa aplicação por exemplo, tem códigos de Content, que são os códigos com o conteúdo da HomePage e dessa página da Aula, e outro código Base, que contém a Sidebar e o Footer.
